# Take-Home Assignment for Course Author job at Boot.Dev

This repository contains two directories that encompass the assets for 2 different lessons that could potentially be part of the `Learn Python` course on [Boot.Dev](https://boot.dev)

## Note

I believe that list comprehensions should be more than 1 lesson. The directory in this repo contains what I think should be the first lesson, which is how to use list comprehension with a for clause. In a subsequent lesson we can introduce the if clause, and then have a third lesson that touches on nested list comprehensions.
